# FootballWorldCupScoreBoard

Includes 3 projects: 
- FootballWorldCupScoreBoard (Console example project),
- FootballWorldCupScoreBoard.Facts (Unit tests project),
- ScoreBoardLibrary (Library project)

# Build project
- Install .Net Framework 4.8
- Set FootballWorldCupScoreBoard as startup project (if needs) or create your own project and use ScoreBoard class in ScoreBoardLibrary.
